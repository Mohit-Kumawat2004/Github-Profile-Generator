## 🧠 Features

✅ Fetches real-time user data from the GitHub API  
✅ Displays essential profile details like:
- 👤 **Name**
- 📍 **Location**
- 🧑‍💼 **Company**
- 📦 **Public Repos**
- 👥 **Followers & Following**
- ✍️ **Bio**
- 🔗 **Blog URL**
- 🖼️ **Profile Avatar**

✅ Responsive Design – works on all screen sizes  
✅ Clean UI with modern styling
